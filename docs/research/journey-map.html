행동 중요 일정이 생길 때마다 카카오 캘린더에 몇 주 전 기록 (7시·1시간 전·15분 전 알림) 전날, 다음날 시간대 계획을 짬(예: 21:00 헬스). 현재는 거의 안 함 — 중요한 것 몇 개만 일정처럼 기록 정한 시각 얼마 전 알림 수신 "가야하나…꼭? (4분) 그래도 기록까지 해놨는데 가야지 (준비) 가기 싫은데…그래도 (2분) 나감" — ~10분 지연 후 실행 운동 감. 지출=결제 직후, 식사=먹은 직후 기록 (칼로리는 종종 까먹어 늦게·부정확) 그 블록을 '완료' 체크 (미실행 시 미실행+사유) 모든 기기 자동 동기화. (나중) 계획 vs 실제·실패 사유를 모아 봄

확인 여정 검증 (init2/instructions 기준)

◐ 여정이 실제 사용 패턴과 맞는가?

대체로 YES — ①·③·④·⑤·⑥은 P1 인터뷰 실측과 일치. 단 ②(D-1 설계)는 현재 하지 않는 단계 — 여정 전체가 아직 습관이 아닌 스텝에 의존한다(=최대 채택 위험).

✔ 불편 지점이 HMW와 연결되는가?

YES — 각 단계 불편이 HMW로 직결. 가장 아픈 ④=H2(투표 1위), ⑤ 마찰=H16/H12, ⑥ 이탈=H7/H11. 표의 'HMW 연결' 행 참조.

✔ 놓치던 문제 지점이 새로 보이는가?

YES, 3가지: ① ②단계 자체가 빠진 습관(전 여정의 관문) → 온보딩/부담↓ 최우선. ② 실패는 ④ 이전 ②(계획 시)에서도 발생("과부하 → 운동 제외") → H3를 계획 시점에 발동해야. ③ ⑤ 기록 마찰의 조용한 2차 하강이 매일-사용을 서서히 갉음.
